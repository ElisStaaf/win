STRUCT Tutorial, your new data manager.

  • Introduction

    Welcome to STRUCT, your new data manager. Have some data to keep track of employees, projects or investors? Don't use CSV, use STRUCT! STRUCT was made in python, and managed in python, so, let's get started!

    Get started

    First, download STRUCT from github. Then, and only then, can you get started. To begin, create a python file. Name it something useful, like index.py and then open it. Then, write the following:

    from structs import StructParser

    NOW, you should create a STRUCT, file, it's generally good practice to name it the same as the python file, so index.struct.

    Structure your database

    Values in STRUCT are separated by ":". The first row should be the category of the values, for example, say that you wan't to manage employees of a small company. You would write this inside of the index.struct file:

    name:department:age
    John:Sales:30
    Jane:Marketing:28
    Mark:Finance:35

    Here, we defined three people with three different attributes: name, department and age.

    Use your STRUCT

    Your database is almost ready! You just need to send your STRUCT to your database! Firstly, have your database ready, for demonstration purposes, it will be named index_db.txt. Have the database ready! This is the updated index.py file:

    from structs import StructParser
    parser = StructParser("index.struct")
    parser.make("index_db.txt")

    This will push the changes you've made to the database! Let's take a look!

    name,John,Jane,Mark
    department,Sales,Marketing,Finance
    age,30,38,35


    Conclusion

    Yeah, so STRUCT is an easy and intuitive open-source data-manager, the YOU, (or your company) should use! Goodbye!
