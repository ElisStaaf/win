MS Guide: By Elis Staaf

MS (MochaScript) is a python-based, high level programming language made by... Me! MS is very similar to python, but has some differences! For example, the syntax is more flexible, but harder to understand. For an example:

Python:
HelloWorld = "Hello World!"
print(HelloWorld)

MS:
str HelloWorld <= "Hello World!"
: HelloWorld

Both of these programs print "Hello World!" to the console. To create an MS script, simply create a file with the ".ms" file extension. Anyways, Let's get onto the tutorial!

Chapter 1: The basics

Now that you've got a basic understanding of the differences of MS and Python, let's get onto our first keyword: python. I know, i know, it's ironic. But MS can actually execute python code, making it a superset of python, kinda?? Anyway, let me show you how to use it. Let's write a simple program using both MS AND the python keyword.

str person <= "Richard"
python def greet(name): print("Hello, %s!" % name)
python greet(person)

Output:
Hello, Richard!

Buuut, if you have no prior programming experience, this is very hard to understand. So let's add some comments,
comments in MS start with "//".

// Defining a variable to represent a person.
str person <= "Richard"

// Defining a function to greet a person, and then calling said function.
python def greet(name): print("Hello, %s!" % name)
python greet(person)

Output:
Hello, Richard!

Nooow, onto our SECOND keyword: ":" (or alternatively print) is the print function in MS. Let's use it!

// Prints "Hello, world!" to the console.
: "Hello, world!"

// ALSO prints "Hello, world" to the console.
print "Hello, world!"

Output:
Hello, world!
Hello, world!

The print function, ofcourse, ALSO works with variables!

// Prints "Hello, Bob!" to the console.
str person <= "Bob"
: person

// ALSO prints "Hello, Bob!" to the console.
str person <= "Bob"
print person

Output:
Hello, Bob!
Hello, Bob!

Making variables in MS is easy, the syntax goes like this:

{type} {name} <= {value}

In MS,assigning values to variables is done with the insert ("<=") operator.
The types in MS basically are just for readability, and they are:

{str}, {int}, {float} and {bool}

Chapter 2: The Novice Level

Nooooooow! New for MS 1.1, functions! You can easily define them by using the "fun" keyword, and end them by using the "end" keyword! The syntax goes like this:

fun {name}
{body}
end

You can call functions by using their name! Let's create a "greet" function by using MS!

// Define a person
str person <= "Bob"

// Define a function to greet a person
fun greet
: "Hello, "
: person
end

// Call said function
greet

Output:
Hello, Bob
